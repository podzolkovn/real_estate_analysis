## Зависимые системы:
- https://online.kompastour.kz/
- PostgreSQL
- DTC.Satourn.CompetitorMonitoring.Service

## Реализованный функционал:
### (Release SAT-0.1)
- Поиск туров по параметрам через интерфейс сайта `https://online.kompastour.kz/search_tour`.
- Сбор данных из:
  - таблиц туров;
  - диаграмм цен.
  
- Получение и сохранение информации о турах:
  - даты и направления туров;
  - цены;
  - отели;
  - авиакомпании.
  
- Учет валют и конвертация цен.
- Обновление данных: если парсер находит меньшую цену для тура — обновляет запись в базе данных.
- Асинхронная обработка задач с использованием Celery.
- Поддержка многопоточной обработки с ограничением количества одновременно выполняемых задач.
- Полная интеграция с базой данных PostgreSQL.

### (Release SAT-0.2)
- Удалена задача по сбору курса валют
- Адаптирован парсер под новую версию Samotour 2025.4
- Оптимизирована логика обработки данных из парсера
